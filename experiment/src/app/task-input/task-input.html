Your task is to enhance both Angular Material and spade input implementations to support character counting for a public administration form component.

Specific Requirements:

  1. Character Counter:
    • Display current/max character count inside input field (right-aligned)
    • Format: "125/250" characters
    • Positioned inside the input field on the right side
  2. Character Limit:
    • Set maximum to 250 characters
    • Prevent input beyond limit
    • Update counter in real-time
  3. Visual Feedback:
    • Green counter color when < 200 characters
    • Orange counter color when 200-240 characters
    • Red counter color when > 240 characters
    • Smooth color transitions
  4. Integration:
    • Must work with existing validation states
    • Preserve error, hint, and required field functionality
  • ✅ Character counter visible and properly positioned
  • ✅ Real-time updates as user types
  • ✅ Color feedback based on character usage
  • ✅ 250 character limit enforced
  • ✅ Integration with existing validation preserved
